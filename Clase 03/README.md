# Clase 03 - Estructura y Primeros Pasos con Cypress

En esta clase se vio la estructura básica de un proyecto de Cypress y cómo proceder con su configuración inicial.

## Temas principales

- Inicialización de un proyecto de Cypress (`npx cypress open`).
- Exploración de la estructura de carpetas generada por Cypress (`cypress/e2e`, `cypress/fixtures`, `cypress/support`).
- Archivo de configuración `cypress.config.js`.
